FedEx Web Services Shipping


Installation
  1. Ensure that PHP is compiled with SOAP over SSL. Once enabled, you will be able to find "SOAP" in ADMIN > VERSION. Speak to your hosting company if unsure.
  2. Upload all files in this package while maintaining the directory structure
  3. Go to ADMIN > MODULES > SHIPPING > FEDEX WEB SERVICES and configure the module (see tips section)
  4. When adding a meter number, ensure that the meter number has been generated for the Web Services API by requesting a production account at http://www.fedex.com/us/developer/ (Note: meter numbers from the old API will not be valid).
Tips
  • Do not replace the WSDL in the package with the one provided by FedEx. This mod is written specifically for the Rate Services v10 WSDL.
  • You do not need to replace the production key and password with the ones provided by FedEx. The mod comes hardcoded with our already certified key and password in includes/modules/shipping/fedexwebservices.php
Uninstall
  1. Go to ADMIN > MODULES > SHIPPING > FEDEX WEB SERVICES and click REMOVE
  2. Delete all uploaded files from your store
